# DuartTech - LandingPage

## Descrição

Projeto de uma LandingPage, com um servidor NodeJS com rotas e página de ERROR 404 caso o usuário entre em uma rota inválida.

## Instalação

```bash
npm install
```

## Uso

```bash
npm start
```

## Estrutura do Projeto

```
DuartTech/
├── public/
├── views/
└── README.md
```
